About this UTM

  • The code and data sections that you input above are merged and encoded into a single string, with '#' as a delimeter.
  • This follows the 'normal' construction of UTMs - The actual machine only accepts a single string, but i've separated these to be more user-friendly.
  • If the machine encounters an illegal transition, it halts.
  • This follows the design of the original Turing thesis.

  • This was whipped up in a day (real fast) to serve for a uni assignment.
  • The actual source code is likely quite bad! You can download it below, but it's messy and not well commented.
  • If you do want to mess with it, you'll probably want to mess with maps.js.
  • maps.js stores the encodings for each symbol, as well as the bitwidth of the state(by default, 4 bits), etc.
